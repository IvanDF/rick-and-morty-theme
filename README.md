## ‚ú® Installation

### Quick Install

1. Open VS Code
2. Go to Extensions (`Cmd+Shift+X` / `Ctrl+Shift+X`)
3. Search for **"Rick and Morty Theme"**
4. Click Install
5. `Cmd+K Cmd+T` / `Ctrl+K Ctrl+T` ‚Üí Select your theme!

### Manual Install

```bash
# Clone the repo
git clone https://github.com/interdimensional-cable/rick-and-morty-theme

# Copy to extensions folder
cp -r rick-and-morty-theme ~/.vscode/extensions/

# Reload VS Code and select the theme!
```
